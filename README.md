# CODSOFT_PP_01--To-Do-List
This project is a simple and user-friendly To-Do List application designed to help users manage their daily tasks efficiently. It allows users to add new tasks, view all existing tasks, update task details, mark tasks as completed, and delete tasks when they are no longer needed. The application focuses on improving productivity by organizing tasks in a clear and structured way. A clean interface with a gray-themed GUI ensures a minimal and distraction-free user experience. This project demonstrates basic CRUD operations, logical flow control, and GUI handling, making it ideal for beginners learning application development.
